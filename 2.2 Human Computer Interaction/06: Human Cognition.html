People
  • We have many interfaces to interact with in the world such as sensing, perception, action and language - many are innate, others are learnt.
  • We form models of the world and use these models to interact with the world and then refine them based on experience. 
  • Gulf-of-Execution: Gap between user’s goal and knowing what actions to take.
Affordances: Actions made possible by an object, users bridge the gulf of execution by learning affordance - central concern is user interface design.
Cognitive
Aspects of UI that supports thinking about possible actions e.g. labelling buttons reflecting which actions it would trigger.
Physical
Aspects of UI that supports physical action e.g. mouse with buttons and scroll wheel.
Functional
Aspects of UI that respond to input to effect some outcome e.g. clicking mouse button within boundaries of ‘add to cart’ icon will add the item to your cart (result).
Sensory
Aspects of UI that help user sense cognitive and physical affordances e.g. highlighting actions that are available.

Multi-Store-Model

Problems related to STM
  1. Limited Capacity: 7+2 Miller, increased through chunking
  2. Volatile: Decay occurs frequently

Examples
  • When giving instructions they must be easy to remember or are shown on screen.
  • Sequential dialog with number of options to remember e.g. voice memos.
  • Forms where later entries depend on what you entered earlier.
  • Complex actions where user needs to remember steps completed and steps to do.
  • Information in different places where user needs to jump and compare.
  • User needs to remember which mode the UI is in.
  • Reaction time so slow you forget what you were looking for.

Recognition
  • We need to activate certain memories to remember how to use certain UI.
  • Triggered by new perception that is similar to something that has been stored.
  • Encountering something and remembering what it is.
  • Affected by context i.e. what we expect to encounter.

Recall
  • Memory ‘look up’ assisted by cues and associations.
  • Cues need to be significant in order to recall memories and form an accurate representation of how to use system.
