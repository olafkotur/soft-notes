User Interfaces vs Programmable Interface
  • No direct manipulation: Modern UIs actions result in immediate feedback.
  • Use of notation: Grace Murray Hopper developed compiler - translator English/Code. 
  • Use of abstractions: Interactive UIs enable us to manipulate concrete items, in programming we have to use abstractions.


History
1968
Doug Engelbart had introduced idea of pointing, fundamental input principle.
1972
Alto: Computer for “Regular Folks” created, Xerox PARC. Ability to. View files, move and edit documents. Included ability to scroll with mouse.
1981
Star: Desktop Interface, Xerox PARC. Included Graphical UI (GUI), each program and document is an object, icons made graphical operations possible - no longer open files using commands. User is able to copy, cut and paste.
