UML Class Notation


Techniques for identifying classes / relationships
  • Grammatical approach: Classes and attributes are nouns (use tangible entities, roles, interactions and locations). Operations are verbs e.g. actions carried out by an object class.
  • Relationships: 
    1. Is the object class a subclass of another: Inheritance
    2. Is the object classes part of another: Aggregation
    3. Does object class interact with another: Relationship
