Identifying Network Process
  • IP address: Network layer identifier using 32 bits for IPv4 and 64 bits for IPv6
  • Port number: Transport layer identifier (16 bits)
  • Many processes can be running on same host, using different ports the OS kernel can forward messages to correct processes
  • Root tends to use 0 - 1023 ports, clients use 1024 - 65535


What does an application need?
Data Integrity
Some apps require 100% reliable data transfer
Throughput
Some apps (multimedia) require minimum through to be effective
Timing
Some apps require low delay to be effective (games, telephone)
Security
Encryption, data integrity
Common applications


TCP Service
  • Reliable transport between sending and receiving
  • Flow control: Sender won’t overwhelm receiver
  • Congestion control: Throttle sender when network overloaded
  • Connection-oriented: Setup required between client and server processes i.e. client TCP establishes connection to server TCP
  • Doesn’t provide timing, min throughput guarantee, security

UDP Service
  • Unreliable data transfer between sending and receiving process.
  • Doesn’t provide anything else mentioned in TCP.



Secure Socket Layer (SSL)
  • Application layer
  • Provides encrypted TCP connection
  • Clear text passwords sent into the socket and then traverse internet encrypted


Application Structure
  • Client-server
    • Server is always on with permanent IP addresses
    • Clients have dynamic IPs and do not communicate directly with each other.
  • Peer to Peer (P2P)
    • No always on server
    • Peers request service from other peers and provide service in return
    • Self scalability as new peers can be added providing more capacity
    • Peers and intermittently connected and change IP thus complex management
    • Example Napster where content is decentralised but location of hosts is centralised



Content Distribution Networks (CDN): Stores copies of content at CDN nodes
  • Subscriber request content from CDN
  • Directed to nearby copy, retrieves content
  • May choose different copy if a network is congested
