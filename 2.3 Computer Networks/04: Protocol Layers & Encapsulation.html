Abstraction of Layering


Benefits of Protocol Layering
  1. Structure: Tells us how to organise functionality without implementation
  2. Extensibility: Module implementations may change
  3. Modularity: Eases maintenance and allows identification of complex system pieces
  4. Flexibility: Reuse of code across the network

Header Encapsulation
  • Each layer is added with a header
  • When processing received information, each header is removed
  • Can cause a lower performance due to the header overhead
