Ideal: Perfect knowledge, sender sends only when router buffer is available.
  • Routers often have finite space, thus if router let's know when they are free it reduces congestion.
  • Router does not have to discard packets when the buffer is free thus there is little to no retransmission.

Realistic: Packets can be lost or dropped at router due to full buffers.
  • Sender can also time out prematurely, sending two copies , both of which are delivered, thus wasting space.
  • If routers have multi hop paths then one packet path might be dropped if another packet path is retransmitted. 



Congestion Control Approaches: Sender increases transmission rate, probing for usable bandwidth until loss occurs.
Additive Increase
Increase cwnd (similar to rwnd) by 1 MSS every RTT until loss detected. 
Multiplicative decrease
Cut cwnd in half after loss.




Slow Start: Send 1 MSS initially and then double each time if none is dropped. 
  • Thus the initial rate is slow but ramps up exponentially fast.
  • Increase switches to linear when reaching ssthresh - initialised to a large value at start then set to half of cwnd when a timeout happens.


Congestion Control Versions
TCP Tahoe
Original version of TCP congestion control, it has a slow start but a fast retransmission rate.
TCP Reno
Same as Tahoe but with Fast Recovery
TCP Vegas
Completely new implementation based on delay variation instead of packet loss.
Fast Retransmit


Fast Recovery: Begins when 3 duplicate ACKs are received. 
  • TCP retransmits missing packet and waits for ACK before returning to Avoidance thus avoiding Slow Start.
  • If no ACK present then return to slow start.
  • In most cases network is only sometimes congested so it's okay to do this.


Congestion Avoidance: The network keeps getting tested to its limits, trying to find out the true limit of network.
  • Conditions keep changing on the network thus limits will be changing.
  • Need to maximise transfer of data over TCP at all times.
  • Avoidance ends when TCP will push the congestion window too far where a loss will occur.
