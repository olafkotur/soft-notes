Major Components

Mail Servers
  • Mailbox contains incoming messages for user
  • Message queue of outgoing mail messages (to be sent)
  • SMPTP protocol between mail servers to send email messages
User Agents
  • Composing, editing, reading mail messages e.g. Gmail
  • Both outgoing and incoming messages stored on server
SMTP
  • Simple Mail Transfer Protocol
  • Delivery and storage to receiver and server
  • Binary multimedia data has to be encoded to ASCII before sending which is then decoded by receiver
  • Uses TCP to transfer email message from client to server (port 25)
  • Command/Response like HTTP - ACSII TEXT / STATUS CODE and PHRASE
  • Requires both header and body to be in 7-bit ASCII
  • Server uses CRLF to determine end of message i.e.Â \r\n

Nature of connection
  • TCP connection is a direct connection between servers regardless of distance
  • Example, Lancaster will be directly connected to New York
  • Different to the nature of routing




Mail Access Protocols
  • POP3: Post Office Protocol - authorisation, user and password.
  • IMAP: Internet Mail Access Protocol
    • Put things into folders server side
    • Allows manipulation of stored messages
    • Maintains states across sessions
    • Allows only parts of messages to be retrieved
  • HTTP: Gmail, Hotmail, Yahoo Mail
