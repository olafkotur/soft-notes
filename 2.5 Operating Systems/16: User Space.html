Creating Process
  • Using fork( ) system call we are able to return child’s Process ID to parent to 0 to child.


Adjacent Faulting: On page fault, we often load adjacent pages as we assume a process will access them soon - we can adjust this to match available memory:
  1. Limited Memory: Load only accessed pages from disks
  2. Beaucoup de Memory: Load adjacent pages e.g. WinNT loads up to 7 pages.


Catching NULL pointer references
  • Processes often will have NULL when they are looking for Page 0
  • Causes page fault for any reference to page 0
  • We can send SIGSEGV to process that has a segmentation fault and core dumped.
