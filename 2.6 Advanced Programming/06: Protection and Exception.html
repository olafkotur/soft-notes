Final: Defines total immutability - can be applied to class, methods and instance variables
  • Defined classes can never be sub-classed
  • Defined methods can never be changed using override
  • Defined variables can never be overwritten

Packages: Groups classes and interfaces together - logical groupings independent of inheritance hierarchy.
// Creating
package LancasterUniversityThings;
class Person {
    public void sayHello() {
        System.out.println(“Hello World”);
    }
}

// Accessing method uno
class Driver {
    public static void main(String[] args) {
        LancasterUniversityThings.Person myPerson;
        myPerson = new LancasterUniversityThings.Person();

        myPerson.sayHello();
    }
}

// Accessing method duo
import LancasterUniversityThings.*;

class Driver {
    public static void main(String[] args) {
        Person myPerson = new Person();
        myPerson.sayHello();
    }
}
