Map( ): Applies function to each element in a collection, takes 3 arguments
  1. Current element
  2. Current index in collection
  3. Current collection
const numbers = [1, -1, 2, 3];
const filtered = numbers.filter(n => n >= 0);

const items = filtered.map(n => ‘<li>’ + n + ‘</li>’);

// Result of items
0: “<li>1</li>”
1: “<li>3</li>”
2: “<li>2</li>”


Reduce( ): Applies function with shared accumulator to each element in collection, takes 2 arguments
  1. Accumulator
  2. Current Value
const numbers = [1, -1, 2, 3];

const sum = numbers.reduce((acc, currentValue) => {
    return acc + currentValue;
});

// Result of sum
5
