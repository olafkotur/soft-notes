Normalisation: Process that takes relation schema through series of tests to certify whether it satisfies certain normal form.
  • Aims to remove Insertion, Update and Deletion dependencies.

First Normal Form
  1. Domain of an attribute must include only atomic values - can’t be divided i.e. ‘green’.
  2. The value of any attribute in a tuple must a single value from the domain of that attribute.
